## KANNAKLAUS CHARACTER

### Description
KannaKlaus is a cannabis-friendly Santa Claus figure. Key characteristics:
- **Outfit:** Forest green Santa suit (NOT red)
- **Trim:** Gold accents
- **Beard:** Traditional white Santa beard
- **Demeanor:** Friendly, welcoming, festive
- **Props:** Santa bag, gift boxes, cannabis-adjacent items

### Usage Guidelines
- Always in green outfit (brand consistency)
- Use for welcoming/CTA slides
- Can hold "Join Us" tag or similar
- Position: Usually bottom-right or center
- Size: 25-50% of frame depending on slide purpose

### DO NOT
- Show KannaKlaus consuming cannabis
- Use red outfit (that's regular Santa)
- Make him look aggressive or unfriendly

## AI IMAGE GENERATION NOTES

### Common Issues to Watch For
1. **Garbled text:** AI often mangles business names - verify spelling
2. **Duplicate entries:** May repeat vendors/sponsors - check count
3. **Wrong panel count:** Spec says 5 vendors, verify 5 panels
4. **Character consistency:** KannaKlaus should stay green-suited

### Quality Checklist
- [ ] All text readable and spelled correctly
- [ ] Correct number of panels/items
- [ ] No duplicates
- [ ] Colors match brand palette
- [ ] KannaKlaus in green (not red)
- [ ] Instagram handles match (@correct_handle)

### Regeneration Protocol
If issues found:
1. Document specific problems
2. Create more explicit prompt with corrections
3. Regenerate only the broken slide(s)
4. Re-verify before marking complete
