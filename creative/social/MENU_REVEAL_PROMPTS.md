## STYLE NOTES

**DO:**
- Keep text legible (will be viewed on mobile)
- Use clean, professional typography
- Include subtle holiday elements (snowflakes, ornaments, gold sparkles)
- Make prices clearly visible
- Use consistent gold/green color scheme
- Add decorative borders and frames

**DON'T:**
- Include food photography
- Use cannabis imagery (leaf, bud, etc.)
- Make it look cluttered
- Use hard-to-read decorative fonts for menu items
- Forget the event date/location
- Make it look too "weed-y" - keep it classy
